## 简介

一款 `OneDrive` 目录文件索引应用，基于优雅的 `PHP` 框架 `Laravel5.8` 搭建，并通过 `Microsoft Graph` 接口获取数据展示，支持多类型帐号登录，多种主题显示，简单而强大。
- 魔改自：[https://github.com/WangNingkai/OLAINDEX](https://github.com/WangNingkai/OLAINDEX)

## 魔改功能

- 获取所有下载链接
